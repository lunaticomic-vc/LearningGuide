# 1. Въведение в изкуствения интелект

## Какво е изкуствен интелект?
Изкуственият интелект (ИИ) е област в компютърните науки, която се стреми да създаде системи, способни да изпълняват задачи, които обикновено изискват човешки интелект.  
Примери:
- разпознаване на образи  
- превод между езици  
- игра на шах или Go  
- генериране на текст (като GPT моделите)

ИИ **не е магия**, а математика + статистика + алгоритми.

---

# 2. Основни категории ИИ

## 2.1 Тесен ИИ (Narrow AI)
Системи, обучени за една конкретна задача.  
Примери:
- Модел за разпознаване на котки  
- Препоръчваща система на Netflix  
- Гласови асистенти като Siri  

## 2.2 Общ ИИ (General AI)
Хипотетичен тип ИИ, способен да изпълнява **всяка** интелектуална задача, която човек може.  
Все още **не съществува**.

## 2.3 Суперинтелигентен ИИ (Superintelligence)
ИИ, който превъзхожда човешката интелигентност във всички аспекти.  
Все още **не съществува**.

# 3. Фундаментални компоненти на ИИ системите

## 3.1 Данни (Data)
ИИ се учи от данни — текст, изображения, аудио, числови таблици.

## 3.2 Модели (Models)
Моделът е „мозъкът“ на системата — математическа функция, която научава структури в данните.

## 3.3 Обучение (Training)
Процесът, при който моделът се учи чрез оптимизация.

## 3.4 Инференция (Inference)
Етапът, в който моделът вече прави прогнози.

# 4. Как работят моделите?  
### Интуиция за абсолютни начинаещи

Представете си модел като функция:

```
вход → модел → изход
```

Пример:

```
„Котка“ → модел → „Животно“
```

Моделите нямат „разбиране“ — те търсят закономерности.

---

# 5. Машинно обучение (Machine Learning)

## 5.1 Надзиравано обучение (Supervised Learning)
Имате вход и правилен отговор.

Пример:  
**Вход:** снимка на животно  
**Етикет:** „котка“

Моделът се учи чрез много примери.

### Минимален пример (Python)

```python
from sklearn.linear_model import LinearRegression
import numpy as np

# Данни: часове учене → резултат на тест
X = np.array([[1], [2], [3], [4]])
y = np.array([50, 60, 70, 80])

model = LinearRegression()
model.fit(X, y)

print(model.predict([[5]]))  # очаквано: ~90
```

## 5.2 Ненадзиравано обучение (Unsupervised Learning)
Моделът открива структури без зададени етикети.

Пример: групиране на клиенти по поведение.

### Пример (K-Means Clustering)

```python
from sklearn.cluster import KMeans
import numpy as np

X = np.array([[1, 2], [1, 3], [10, 10], [11, 11]])

kmeans = KMeans(n_clusters=2)
kmeans.fit(X)

print(kmeans.labels_)
```

# 6. Дълбоко обучение (Deep Learning)

Дълбокото обучение използва *невронни мрежи* — математически структури, вдъхновени от мозъка.

### Много опростена схема

```
Вход → [Слой 1] → [Слой 2] → [Слой 3] → Изход
```

Всеки слой променя данните чрез матрици и нелинейности.

### Мини пример с PyTorch

```python
import torch
import torch.nn as nn

# Проста мрежа: вход 2 → скрит слой 4 → изход 1
model = nn.Sequential(
    nn.Linear(2, 4),
    nn.ReLU(),
    nn.Linear(4, 1)
)

x = torch.tensor([1.0, 2.0])
print(model(x))
```

# 7. Генеративни модели (Generative AI)

Това са модели, които **създават нови данни**:

- текст (GPT)
- изображения (Stable Diffusion)
- музика
- код

### Как работят?
Генеративните модели предсказват „следващия елемент“:
- следващата дума  
- следващия пиксел  
- следващия токен  

### Пример за генерация на текст (pseudo-code)

```python
text = "Днес времето е"
next_word = model.predict_next(text)
print(next_word)
```

# 8. Ограничения и грешки на ИИ
ИИ прави грешки:
- халюцинации (измислени факти)
- пристрастия (bias)
- чувствителност към входа
- липса на разбиране

Оттук идват важни области като **AI safety**, **fairness**, **robustness**.

# 9. Пример: Мини-класификатор от нулата (с numpy)

```python
import numpy as np

# Примерна логистична регресия (много опростена)
def sigmoid(z):
    return 1 / (1 + np.exp(-z))

# Х = часове учене, Y = 1 ако е минал изпита, иначе 0
X = np.array([1, 2, 3, 4])
Y = np.array([0, 0, 1, 1])

w = 0.1
b = 0.1
lr = 0.1

for _ in range(1000):
    z = w * X + b
    pred = sigmoid(z)

    dw = np.mean((pred - Y) * X)
    db = np.mean(pred - Y)

    w -= lr * dw
    b -= lr * db

print("Вероятност да мине с 5 часа учене:", sigmoid(w * 5 + b))
```

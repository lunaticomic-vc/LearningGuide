# Основи на изкуствения интелект (Foundations of AI)

## 1. Въведение
Изкуственият интелект (ИИ) е област от компютърните науки, която се стреми да създава системи, способни да изпълняват задачи, които обикновено изискват човешки интелект — като разпознаване на образи, вземане на решения, разбиране на език и учене от опит.

Този документ е базовият материал на курса и е създаден за абсолютни начинаещи.

---

## 2. Видове изкуствен интелект

ИИ обикновено се дели на три основни категории:

### 2.1 Тесен (Narrow AI)
Това са системи, които могат да изпълняват **само една конкретна задача**.
Примери:
- GPT, който пише текст
- Система за препоръки в YouTube
- Класификатор на снимки

Тесният ИИ е единственият, който реално съществува днес.

### 2.2 Общ (General AI)
Общ ИИ е система, която може да разбира, учи и решава различни задачи **толкова добре или по-добре от човек**.

Такъв ИИ не съществува, но се изследва активно.

### 2.3 Свръхинтелигентен (Superintelligent AI)
ИИ, който е по-умен от всички хора във всички области.  
Това е хипотетично ниво и основен фокус в областта на AI Safety.

## 3. Как машините учат

ИИ използва различни методи за учене. Ето основните:

### 3.1 Модели и данни
Моделите са математически функции.  
Данните „учат“ модела да разпознава връзки.

Пример:  
Даваме на модела хиляди снимки на котки → той научава кои пиксели и форми са типични за котка.

### 3.2 Примерен Python код: проста „моделна“ логика

```python
# Пример: проста функция, която учи модел чрез правила.
# Това не е истински ML модел, но обяснява идеята за "pattern recognition".

def is_cat(description):
    keywords = ["козина", "мустаци", "мяу", "опашка"]
    score = sum(1 for k in keywords if k in description.lower())
    return score >= 2

print(is_cat("Животно с козина и мустаци"))   # True
print(is_cat("Голямо животно с рога"))        # False
```

## 4. Как работят невронните мрежи (интуитивно)

Не е нужно да сте добри по математика, за да разберете концептуално:

1. Мрежата приема входове (например пиксели).
2. Умножава ги по числа („тегла“).
3. Сумира ги.
4. Прилага функция.
5. Прави прогноза.

Тя **не мисли**, а **пресмята**.  
Но чрез много преповтаряне и данни започва да разпознава модели.

Прост пример:

```python
# Една "невронка" (един неврон)
def neuron(x, weight, bias):
    return x * weight + bias

print(neuron(3, 0.5, 1))  # 2.5
```

## 5. Генеративен ИИ (Generative AI)

Генеративният ИИ създава **ново съдържание**:
- текст
- изображения
- музика
- код

Той не „мисли“.  
Той **предсказва следващата дума/пиксел**.

Пример с GPT-подобна логика:

```python
import random

words = ["котката", "спи", "на", "дивана"]
def tiny_generator():
    return random.choice(words)

sentence = " ".join(tiny_generator() for _ in range(5))
print(sentence)
```

## 6. Защо ИИ прави грешки

ИИ греши, защото:
- няма разбиране за света
- работи чрез вероятности
- може да „халюцинира“
- не проверява фактите (освен ако не е свързан с инструмент)

---

## 7. Етика, рискове и AI Safety (базово)

Основни проблеми:
- пристрастия в данните (bias)
- дезинформация
- вредно съдържание
- злоупотреба с инструменти
- автономно поведение на агентни системи

Целта на AI Safety е да се намали рискът и да се подобри контрола.

## 8. Мини-проект за начинаещи: прост чатбот

```python
# Мини чат бот
questions = {
    "как се казваш": "Аз съм учебен ИИ бот.",
    "какво е изкуствен интелект": "Система, която изпълнява задачи като човек.",
}

while True:
    user = input("Ти: ").lower()
    if user == "изход":
        break

    found = False
    for q, a in questions.items():
        if q in user:
            print("Бот:", a)
            found = True
            break

    if not found:
        print("Бот: Все още не знам как да отговоря на това.")
```
